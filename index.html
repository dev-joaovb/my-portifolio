Meus Trabalhos

Engenharia de Software e Desenvolvimento de sistemas

Ao longo da minha jornada, tenho desenvolvido projetos que equilibram estética moderna e arquitetura robusta. Meu portfólio reflete a criação de aplicações escaláveis, desde interfaces front-end responsivas até sistemas de back-end complexos e seguros. Utilizo tecnologias de ponta como React, Node.js e Tailwind CSS, também com ferramentas de IA como N8n e de Low/No Code como Wordpress, Elementor e WooCommerce para garantir que cada solução seja otimizada para alta performance e excelente experiência do usuário. Meu foco é traduzir requisitos de negócio em código limpo e funcional, sempre priorizando a agilidade e a entrega de valor real em cada linha de software que escrevo.
