Snake Game

Um jogo clássico de Snake desenvolvido em HTML, CSS, JavaScript e Node.js.

Sobre o Projeto

Este projeto do Snake Game é uma releitura moderna do clássico atemporal que marcou a história dos jogos digitais. O foco principal do desenvolvimento foi a implementação de uma lógica de algoritmos precisa para o controle de movimento e detecção de colisões em tempo real. Utilizando JavaScript puro para a manipulação da lógica e renderização, o jogo entrega uma experiência de alta fidelidade, garantindo fluidez e uma jogabilidade responsiva que desafia a agilidade do usuário.

A arquitetura da aplicação é potencializada pelo Node.js no back-end, permitindo o gerenciamento de um ranking global de pontuações e a persistência de dados dos jogadores. A interface, construída com HTML5 e CSS3, segue uma estética minimalista e totalmente responsiva, adaptando-se a diferentes tamanhos de tela. Este projeto demonstra minha capacidade de estruturar sistemas interativos complexos e aplicar conceitos de programação orientada a eventos em um ambiente web performático.
